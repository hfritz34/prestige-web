# Prestige Web - Frontend

A modern React frontend for Prestige, a Spotify music analytics dashboard.

## Features
- 🎵 Real-time music analytics
- 📊 Beautiful data visualizations  
- 🔐 Auth0 authentication
- ⚡ Built with React + TypeScript + Vite

## Tech Stack
- React 18
- TypeScript
- Vite
- Auth0
- Axios
